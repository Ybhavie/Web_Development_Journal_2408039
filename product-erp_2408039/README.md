Product ERP System
A robust, full-stack Spring Boot application designed to manage product inventory efficiently. This project demonstrates the implementation of a RESTful API with a clean 3-tier architecture and a responsive frontend.

Features
Full CRUD Operations: Create, Read, Update, and Delete products through a web interface.
Live Dashboard: Real-time stock management with a modern CSS UI.
RESTful API: Clean endpoints for integration with other services.
In-Memory Database: Powered by H2 for rapid development and testing.

Tech Stack
Backend: Java 17+, Spring Boot 3.x
Data Access: Spring Data JPA (Hibernate)
Database: H2 (In-Memory)
Frontend: HTML5, CSS3, JavaScript (Fetch API)
Utilities: Lombok (to reduce boilerplate code)

How to Run
Clone the Repository: git clone https://github.com/Ybhavie/product-erp-springboot.git

Run the Application: Open in VS Code/IntelliJ and run ProductErp2408039Application.java.

Access the ERP: Go to http://localhost:8080/index.html in your browser.

Database Console: Access the H2 Console at http://localhost:8080/h2-console using JDBC URL: jdbc:h2:mem:productdb
