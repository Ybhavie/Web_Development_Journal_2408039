# 📖 Web Development Journal

Welcome to my Web Development Journal! This folder is a chronological record of my coding journey, class assignments, and personal explorations in web development during my time at **MES Vasant Joshi College of Arts and Commerce**. 

Rather than just uploading finished projects, I am using this space to document my learning process, track bugs I've squashed, and save useful code snippets.

## 🎯 Purpose
* **Track Progress:** Watch my skills grow from basic HTML to complex, interactive web applications.
* **Organize Coursework:** Keep lab assignments, homework, and lecture notes neatly separated by topic/week.
* **Reflect & Learn:** Document the "why" and "how" behind the code I write, including challenges faced and how I solved them.

## 📂 Folder Structure
Here is how this journal is organized:

* `📁 Week_01-HTML_Basics/` - Introduction to semantic HTML and document structure.
* `📁 Week_02-CSS_Styling/` - CSS fundamentals, Flexbox, Grid, and responsive design.
* `📁 Week_03-JS_Fundamentals/` - Variables, loops, DOM manipulation, and event listeners.
* `📁 Mini_Projects/` - Small weekend projects and experiments outside of standard homework.
* `📁 Resources/` - Helpful cheat sheets, links to tutorials, and study notes.
