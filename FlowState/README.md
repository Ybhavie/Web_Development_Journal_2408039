FlowState — Minimalist Task Manager

FlowState is a high-performance, minimalist task management application built with 100% Pure Vanilla JavaScript. Designed for deep work and maximum focus, it offers a distraction-free environment for capturing and organizing objectives with zero-latency persistence.

Core Features
Pure Vanilla Engine: Built entirely with native Web Standards. Zero frameworks, zero libraries, and zero external dependencies for maximum speed and portability.

Intelligent Priority System: Seamlessly categorize tasks into High, Medium, or Low priority tiers to streamline your workflow.

Zero-Latency Persistence: Leverages localStorage for instant data serialization, ensuring your tasks are saved and loaded without server requests.

Real-Time State Management: A robust data-handling logic that updates the UI instantly as tasks are added, toggled, or deleted.

Glassmorphism Architecture: A sleek, modern interface utilizing CSS backdrop filters and recursive shadows for a premium, focused aesthetic.

Synthesized Audio Feedback: Utilizes the Web Audio API to generate real-time auditory cues, providing sensory confirmation for every action.

Technical Implementation
HTML5: Semantic structure designed for accessibility and clean DOM mapping.

CSS3: Custom properties (CSS Variables), Flexbox, and hardware-accelerated animations.

JavaScript (ES6+): Advanced DOM manipulation, JSON serialization, and Event Delegation.
